# ShortenerUrl

### Test URL Shortener
# New Features!

  - api in nodejs
  - BD MongoDB
  - front Angular
  - needed docker for raise bd
### First up bd

```sh
$ cd bd
$ docker-compose up -d
```

### Second up back

```sh
$ cd backend
$ npm i
$ npm start
// puerto 3003
```

### Third up front

```sh
$ cd front
$ npm i
$ ng serve
```

#### open browser localhost:4200
